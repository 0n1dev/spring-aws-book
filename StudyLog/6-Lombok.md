# Lombok
---

> 자바 개발자들의 필수 라이브러리 롬복

- Getter, Setter, 기본생성자, toString 등을 애너테이션으로 자동 생성

## 의존성 추가
---

```java
compile('org.projectlombok:lombok')
```

## 플러그인 추가
---

> IntelliJ 기준

1. 컨트롤 + 쉬프트 + A
2. Plugins
3. Lombok 검색
4. IDE Restart
5. Settings > Build > Compiler > Annotation Processors
6. Enable annotation processing 체크
